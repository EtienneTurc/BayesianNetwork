Bayesian network

Get started

Add nodes by double clicking.
Draw edges by pressing shift and clicking on the nodes.
Delete an edge or a node by pressing delete.
Compute the probabilities by clicking the Compute button.
Save your network by simply clicking on the Save button.
Use a previous network by loading it.

Drag and drop a json file of your network or
select one.

Your file should have the following format:

{
	"nodes": [{
		"id": Int,
		"x": Int,
		"y": Int,
		"name": String,
		"value": Int,
		"parents": Int[],
		"proba": Int[][]
	}]
}
